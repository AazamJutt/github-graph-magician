GitHub Graph Magician

Give a fun look to your Github Contribution Graph

You need min 695px screen to use this app

Step 1. Install Node JS on your pc
Step 2. Create an empty private repository (any name e.g. test) in GitHub and clone it to PC.
Step 3. Download gh_graph_magician and extract all of its contents to cloned repository directory
Step 4. Create pattern in grid below and click Generate button to download the targetSpot.json file



Step 5. Copy targetSpot.json to the directory where you cloned repository
Step 6. Open the terminal (or cmd), navigate to app directory in cloned repository directory and run:

node gh_graph_magician.js

AND see the Magic!

*Run this command again and again to gain more darker shade!
